# Chess 16x16 Online

Простой онлайн-проект на Vue 3 + Node.js для шахмат 16x16.

## Что реализовано

- лобби со списком комнат;
- создание игры с названием, паролем и контролем времени;
- подключение к открытой комнате из списка;
- онлайн-синхронизация через Socket.IO;
- доска 16x16;
- базовые шахматные правила для фигур, очередь хода, проверка шаха и мата;
- победа по мату, времени, сдаче или отключению соперника.

## Принятая расстановка варианта

В проекте используется полная первая линия по вашей схеме.

- Белые слева направо: `R R N N B B Q Q K P B B N N R R`
- Черные слева направо: `R R N N B B P K Q Q B B N N R R`
- На второй линии у каждой стороны стоят 16 пешек.

## Запуск

```bash
npm install
npm run dev
```

После запуска:

- клиент: `http://localhost:5173`
- сервер: `http://localhost:3001`

## Production build

```bash
npm run build
npm start
```
